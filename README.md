# Music Match 🎵

A modern dating website that connects people based on their music preferences and tastes.

## Features

- **Profile Creation**: Create a detailed profile with personal information and music preferences
- **Music Matching**: Advanced algorithm that matches users based on:
  - Favorite music genres
  - Favorite artists
  - Favorite songs
  - Music taste descriptions
- **Compatibility Scoring**: Get a percentage match score with potential partners
- **Responsive Design**: Works beautifully on desktop and mobile devices

## How It Works

1. **Create Your Profile**: Fill out your basic information and music preferences
2. **Get Matched**: Our algorithm finds compatible users based on shared musical interests
3. **Connect**: View detailed profiles of your music matches and start connecting

## Technology Stack

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **JavaScript (ES6+)**: Interactive functionality and matching algorithm
- **Local Storage**: Client-side data persistence

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Create your profile and start finding music matches!

## Matching Algorithm

The compatibility score is calculated based on:
- **Genres (40%)**: Shared favorite music genres
- **Artists (35%)**: Common favorite artists
- **Songs (15%)**: Matching favorite songs
- **Age (10%)**: Age compatibility factor

Higher scores indicate better musical compatibility!
